# Telegram Bot

This is a simple Telegram bot built using the `python-telegram-bot` library. The bot provides various functionalities such as managing products, viewing account information, and more. It is designed to be a practice project and is not yet fully functional for real-world use.

## Features

- **Start Command**: Displays a welcome message and a menu with options.
- **Products Command**: Allows users to add or view products.(currently non-functional)
- **Account Command**: Provides information about the user's account (currently non-functional).
- **Analysis Command**: Intended to provide sales analysis (currently non-functional).
- **Info Command**: Provides information about the project.
- **Help Command**: Offers support information.
- **Settings Command**: Allows users to adjust settings (currently non-functional).

## Prerequisites

- Python 3.8 or higher
- `python-telegram-bot` library
- `pyodbc` for database connectivity
- `python-dotenv` for environment variable management
- a bot token from BotFather
- remember to give your bot an ID to actualy use it!

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/TelegramBot.git
   cd TelegramBot
2. Install the required dependencies:
     pip install -r requirements.txt
3. add  your environment variables to .env file
4. run the bot:
     python main.py
Usage
  Start the bot by sending the /start command in Telegram.

  Use the inline buttons to navigate through the bot's features.
